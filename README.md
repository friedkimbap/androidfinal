# 안드로이드 프로그래밍 기말 프로젝트

#### 학번 : 2071173

#### 이름 : 김영민

#### 분반 : C

#### 주제 : 알람 어플리케이션

## 기능

1. 알람을 해제하기 위해서 2가지의 방법을 구현함. (간단한 수학 문제를 풀어서 해제, 타자를 쳐서 해제)
2. 각 알람에 스위치가 있어서 알람을 켜고 끌 수가 있음.
3. 각 알람은 길게 눌러서 삭제를 할 수가 있음.
4. 알람의 정보들을 하나의 txt파일로 기록하고 그 txt파일을 통해서 알람에 대한 정보를 불러옴.
